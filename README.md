# 専属ひしょ子ちゃん

- Ruby version: 3.2.2
- Rails version: 7.1.3
- Database: PostgreSQL

## 概要

- 毎日の収支管理アプリ。
- フードとドリンクのジャンルと価格を登録。Menu のジャンル：値段のオーダー数から、ジャンル、価格帯の商品の売れ行きを自店分析ができる。
- 毎月の売上と経費、利益をカレンダー表示。
- スマホで使用することを想定して PWA 化。
